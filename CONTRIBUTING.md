# Contributing to StreamFlow

Terima kasih sudah tertarik untuk berkontribusi ke projek StreamFlow!

## Cara Berkontribusi

1. **Fork** repository ini
2. Buat **branch baru** untuk fitur/perbaikan kamu
   ```bash
   git checkout -b fitur-baru
   ```
3. Lakukan perubahan dan **commit**
   ```bash
   git commit -m "Menambahkan fitur baru"
   ```
4. **Push** ke branch kamu
   ```bash
   git push origin fitur-baru
   ```
5. Buat **Pull Request** ke branch `main`

## Proses Review

- Semua Pull Request akan di-review oleh author terlebih dahulu
- Kode akan dicek kualitas dan kompatibilitasnya
- Jika tidak ada masalah, author akan merge ke branch `main`
- Jika ada revisi yang diperlukan, akan diinformasikan di PR

## Panduan

- Pastikan kode sudah ditest sebelum submit PR
- Gunakan bahasa Indonesia atau English untuk commit message
- Jelaskan perubahan yang dilakukan di deskripsi PR

## Pertanyaan?

Jika ada pertanyaan, silakan buka [Issue](https://github.com/bangtutorial/streamflow/issues) baru.
